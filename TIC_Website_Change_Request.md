# TIC Website Change Request
**Website:** https://tiic-swart.vercel.app/
**Prepared For:** Agentic Developer
**Source Documents:** TIC Website Development Guide, Visual Mock-up Description, and Content Transcripts

---

# Executive Summary

Transform the current TIC website into a permanent, multi-edition, NCDMB-branded platform that supports annual programme cycles (2026, 2027, 2028+) with strong content management, archive functionality, programme tracking, application workflows, media galleries, and integration with the Tech-IDEAS / ERIN ecosystem.

---

# 1. Global Website Requirements

## Branding

### Primary Brand
- NCDMB branding must be dominant.
- Use NCDMB colour palette, typography, tone, and layout principles.
- NCDMB logo should be the primary logo across the website.

### Secondary Brand
Display:

> Powered by: Entrepreneurship & Innovation Centre Ltd (EICL)

Placement:
- Footer
- About Page
- Partners Section
- Select banners (subtle)

### Annual Structure
- TIC logo must remain year-agnostic.
- Edition branding should be content/theme based.
- Support:
  - TIC 2026
  - TIC 2027
  - TIC 2028
  - Future editions
